العنوان: القسم 6، الدرس 30: فهم العبارات الشرطية في بايثون
مرحبًا بكم، طلابنا الأعزاء، في محاضرتنا عن العبارات الشرطية في بايثون. سنستكشف اليوم استخدام عبارات 'if'، 'elif'، و'else' من خلال سلسلة من الأمثلة. دعونا نحلل الكود ونفهم كل مفهوم خطوة بخطوة.

عبارة 'if' الأساسية:

x = 30
y = 21
if x > y:
    print("true")
هنا، نستخدم عبارة 'if' بسيطة. تتحقق مما إذا كانت x أكبر من y. بما أن 30 بالفعل أكبر من 21، فإن الشرط صحيح، وسيتم طباعة "true".

عبارة 'if' مع قيم حرفية:

if 5 < 10:
    print("true")
يوضح هذا المثال أنه يمكننا استخدام قيم حرفية في شروطنا. بما أن 5 أقل من 10، سيتم طباعة "true".

عبارة 'if' مع شرط خاطئ:

if 5 < 3:
    print("true")
في هذه الحالة، 5 ليست أقل من 3، لذا فإن الشرط خاطئ. نتيجة لذلك، لن تتم طباعة أي شيء. يتم تخطي الكود داخل كتلة 'if'.

تقديم 'else':

x = 30
y = 21
if x < y:
    print("true")
else:
    print("false")
هنا، نقدم عبارة 'else'. إذا كان الشرط (x < y) خاطئًا، وهو كذلك في هذه الحالة، يتم تنفيذ الكود في كتلة 'else'. لذلك، سيتم طباعة "false".

'if-else' مع السلاسل النصية:

name = "ahmed"
if name == "ahmed":
    print("true")
else:
    print("false")
يوضح هذا المثال أنه يمكننا استخدام مقارنات السلاسل النصية في شروطنا. بما أن الاسم هو "ahmed"، سيتم طباعة "true".

'if-else' مع سلسلة نصية مختلفة:

name = "jermain"
if name == "ahmed":
    print("true")
else:
    print("false")
الآن، الاسم هو "jermain"، لذا فإن الشرط خاطئ، وسيتم طباعة "false".

تقديم 'elif':

name = "ahmed"
if name == "ahmed":
    print("true")
elif name == "sara":
    print("the name is sara")
else:
    print("false")
هنا، نقدم 'elif' (else if). يسمح لنا بالتحقق من شروط متعددة. في هذه الحالة، بما أن الاسم هو "ahmed"، سيتم طباعة "true".

'elif' في العمل:

name = "sara"
if name == "ahmed":
    print("true")
elif name == "sara":
    print("the name is sara")
else:
    print("false")
الآن، الاسم هو "sara". الشرط الأول خاطئ، لكن الشرط الثاني (في 'elif') صحيح، لذا سيتم طباعة "the name is sara".

'if-elif-else' مع شرط غير مطابق:

name = "mahmoud"
if name == "ahmed":
    print("true")
elif name == "sara":
    print("the name is sara")
else:
    print("false")
في هذا المثال الأخير، الاسم لا يطابق أيًا من الشروط في 'if' أو 'elif'. لذلك، يتم تنفيذ الكود في كتلة 'else'، ويتم طباعة "false".
توضح هذه الأمثلة قوة ومرونة العبارات الشرطية في بايثون. فهي تسمح لكودك باتخاذ القرارات وتنفيذ كتل مختلفة بناءً على شروط مختلفة. تدرب على هذه المفاهيم، وستتمكن من إنشاء برامج أكثر تعقيدًا وذكاءً!
